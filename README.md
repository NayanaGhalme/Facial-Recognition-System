# 🔍 M-Secure Facial Recognition System

## 📌 Project Description
AI-based system to detect missing persons using facial recognition.

## 🚀 Features
- User Login & Registration
- Lost Person Registration
- Face Detection using Deep Learning
- Email Alert System
- Location Tracking
- Live Monitoring Dashboard

## 🛠 Technologies Used
- Python
- Flask
- OpenCV
- FaceNet + ArcFace
- SQLite

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
